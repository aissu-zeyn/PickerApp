```bash
git clone https://github.com/aissu-zeyn/picker-app.git
```

2. Navigate to the project directory:
```bash
cd picker-app
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
```
